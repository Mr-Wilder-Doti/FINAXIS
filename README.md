# FINAXIS - Recommendation Robot

Estrutura base inicial do recommendation robot seguindo arquitetura em camadas:

- `recommendation_robot/application`: casos de uso
- `recommendation_robot/domain`: entidades e contratos
- `recommendation_robot/infrastructure`: implementações concretas
- `recommendation_robot/interfaces`: bordas de entrada/saída (API)
- `recommendation_robot/config`: configurações
- `tests`: testes automatizados
